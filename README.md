# Jolocom Interview React Native

## Introduction

This is a react native app built to meet the specification for interview tasks
for Jolocom React Native Developer position

## Requirements

- React Native
- Yarn
- Cocoapods
- Xcode
- Android Development Environment
- Node

## Libraries Used

- `react-native-reanimated`
- `react-native-vector-icons`
- `formik`
- `yup`

## Get Started

### To run the project:

- Clone the repo: `git clone repo-link`
- Install dependencies: `yarn install`
- Install pods `cd ios && pod install && cd ..`
- Run on iOS: `yarn ios` or Run on Android: `yarn android`
- Run tests: `yarn test`

Thank you!
